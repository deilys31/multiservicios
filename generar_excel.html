📊 Generador de productos.xlsx

Esta herramienta crea el archivo productos.xlsx de ejemplo con 5 productos ficticios. Ábrela en un navegador con acceso a internet, haz clic en el botón y mueve el archivo descargado a la carpeta raíz del sitio.

¿Qué hacer después?

  1. Mueve productos.xlsx a la carpeta raíz del sitio (junto a index.html).
  2. Abre el sitio con Live Server en VS Code (clic derecho → Open with Live Server).
  3. El catálogo se cargará automáticamente desde el Excel.
  4. Para agregar productos: edita el Excel y agrega filas. Recarga la página.
  5. Para agregar imágenes: copia los archivos .jpg o .png a productos-imagenes/ y escribe el nombre exacto en la columna imagen del Excel.
